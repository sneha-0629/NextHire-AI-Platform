🚀 NextHire AI Platform
AI-Powered Interview Preparation & Candidate Evaluation System (MERN + Gemini AI)

📌 Project Overview
NextHire AI Platform is a smart interview preparation web application that helps students and job seekers prepare for technical interviews using Artificial Intelligence.

The platform generates AI-based interview questions, evaluates candidate answers, and provides a structured way to practice real interview scenarios.

This project is built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) and integrated with Google Gemini AI to deliver intelligent and dynamic interview experiences.

✨ Key Features
🤖 AI-generated interview questions (technical + HR)

🧠 Intelligent answer evaluation using Gemini AI

👤 Secure user authentication (Login / Register / Logout)

📊 Personalized interview practice experience

💬 Real-time question-answer interaction

🔐 JWT-based authentication system

🌐 Fully responsive modern UI

📁 MongoDB database integration

🧠 Problem Statement
Many students face difficulty preparing for real technical interviews due to the lack of personalized guidance and real-time feedback. Traditional preparation platforms provide static questions but do not evaluate answers intelligently.

NextHire AI solves this problem by providing an AI-powered mock interview platform that dynamically generates questions and evaluates user responses just like a real interviewer.
